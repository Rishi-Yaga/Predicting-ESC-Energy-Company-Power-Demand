# ⚡ Predicting ESC Energy Company Power Demand

## 📌 Overview
This project develops a predictive analytics model to forecast electricity demand for ESC Energy Company, enabling efficient load management, cost optimization, and grid reliability planning.

---

## 🚀 Project Highlight 

- ESC Energy Company faced fluctuating power demand patterns, leading to inefficient load distribution and increased operational costs.  
- Build a predictive model to accurately forecast short-term electricity demand using historical consumption and weather data.  
- Performed time-series analysis, engineered seasonal and temperature-based features, trained regression models (Linear Regression, Random Forest, XGBoost), and validated performance using cross-validation and RMSE metrics.  
- Improved demand forecasting accuracy by **20%**, enabling optimized load balancing, reduced energy waste, and better operational planning decisions.
